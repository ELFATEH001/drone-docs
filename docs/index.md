# Welcome to the Drone Project Documentation

This project was initiated in 2025 by students of the Control & Robotics program
at Université Claude Bernard Lyon 1.

With no constraints other than time, our goal was simple: push the boundaries of
what we could build, and experiment hands-on with state-of-the-art techniques and
algorithms developed for autonomous drones.

---

This documentation walks you through the full journey — where we started, what we
built, the problems we solved, and the ones we didn't. And for everything time
didn't allow us to finish, you'll find the perspectives and directions we hoped to
explore.

**To whoever picks this up next — the project is yours now. Build further.**
